# UpdatedRooms
SCP: CS plugin

Данный сборник плагинов улучшает некоторые комнаты, а именно:
* EZ_Lockroom (локрум) - добавляется отравляющий газ, двери (которые могут блокироваться на время) и кнопки;
* EZ_Medibay (медпункт) - добавляется стаканчик и оружейный шкаф с SCP-500 и SCP-420-J;
* PD (карман деда) - теперь нет постоянных выходов, а предметы, находящиеся в этой комнате, со временем телепортируются в комнаты ТЗС;
* AW (шахта боеголовки Альфа) - если слишком долго находиться в этой комнате, то вас начнёт дамажить.

Некоторые параметры можно менять через конфиг:
* `updated_rooms` - **обязательно должно быть true**, чтобы комнаты вообще улучшились (по умолчанию: false);
* `pd_exit_count` - чем больше значение, тем больше шанс сбежать (от 0 до 8) (по умолчанию: 2);
* `pd_damage_per_second` - сколько урона будет наноситься в секунду в измерении деда (по умолчанию: 3);
* `hd` - спустя сколько секунд нахождения игрока в шахте боеголовки будет наноситься урон (по умолчанию: 45).
